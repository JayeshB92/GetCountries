Get countries

Get world countries in MySQL/Firebird/XML format

This script will generate MySQL or Firebird or XML "Countries" table structure depending on what fields you select. All data is fetched from geonames.org

MySQL Example code

CREATE TABLE IF NOT EXISTS `countries` (
    `idCountry` int(5) NOT NULL AUTO_INCREMENT,
    `countryCode` char(2) NOT NULL DEFAULT '',
    `countryName` varchar(80) NOT NULL DEFAULT '',
    PRIMARY KEY (`idCountry`)
) ENGINE=MyISAM DEFAULT CHARSET=utf8 AUTO_INCREMENT=251;
